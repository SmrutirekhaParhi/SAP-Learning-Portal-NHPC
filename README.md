# SAP Learning Portal
**LMS Internship Project**

## Project Overview
The development of the SAP Learning Portal came to fruition as a part of my internship with **NHPC Ltd.**, and was created with the goal to connect and provide SAP training opportunities for employees at **NHPC Ltd**. The SAP Learning Portal provides a controlled environment for organizing training modules, monitoring trainee progress, and improving the overall education experience for each user working in the organization.

## Tech Stack
- **Back-End:** PHP / CodeIgniter 4
- **Database:** MySQL
- **Front-End:** HTML5 / CSS3 / JavaScript / Other Web Technologies
- **Hosting Environment:** XAMPP / Apache

## Major Features
- Central Learning: Users can access all SAP training materials in one dashboard.
- Restricted Access and Authentication for Trainees

## Project Structure
- `/app/` - Contains all core logic, controls, and models.
- `/public/` - Contains all assets (css/js) and is the main entry point to the application.
- `/tests/` - Contains unit test and functional test cases.
